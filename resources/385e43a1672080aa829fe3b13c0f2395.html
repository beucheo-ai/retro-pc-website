VS Code와 클로드 코드가 세팅되지 않은 분만 열어주세요!

⚙️ 사전 준비

1. VS 코드

1-1. Visual Studio Code 설치 및 클로드 코드 추가

VS 코드 다운로드 링크 : https://code.visualstudio.com/download
VS 코드 실행 후 좌측 네모 아이콘 클릭 → 클로드 코드 설치하기 → 클로드 유료 계정 로그인까지 해주세요.

1-2. 클로드 코드 실행하기

1. 세팅은 딱 한 번만 (제일 쉬운 부분)

영상 만드는 도구는 하이퍼프레임(HyperFrames) 이라는 무료 오픈소스예요. 직접 설치 명령어 칠 필요 없이, 클로드 코드한테 링크 주면 알아서 깔아줍니다.

클로드 코드 채팅창에 이렇게 입력해주세요:

https://github.com/heygen-com/hyperframes
이거 설치해줘. 영상 만들 거야.

2. 먼저, 딱 3가지만 준비하세요 (준비물)

AI가 알아먹는 재료 3가지만 챙기면 끝이에요.

📂
시작 전 1분 세팅

바탕화면에 폴더를 하나 만드세요 (예: 영상제작). 앞으로 준비하는 파일은 전부 이 폴더 안에 넣습니다 — AI가 이 폴더를 들여다보고 영상을 만들거든요.

① 내용 원고 — .md 파일

영상에 들어갈 글이에요. 마크다운(.md) 형식으로 준비합니다.

💡
왜 마크다운(.md)일까요?

AI가 제목·소제목·표를 읽고 '장면(씬)'을 자동으로 나눠주기 때문이에요.

📝 만드는 법

  1. 노션에 제목·소제목·내용을 구분해서 글을 씁니다.
  2. 우측 상단 ••• → 내보내기 → Markdown 선택 → .md 파일 완성.
  3. 그 파일을 만든 폴더에 넣습니다.
채팅창에 내용을 바로 적어도 되지만, 이렇게 구조를 잡으면 장면 구분이 훨씬 깔끔해서 추천해요.

② 디자인 — design.md 파일

영상의 색·폰트·디자인 규칙이에요. 이게 있으면 AI가 매번 색을 새로 지어내지 않고 일관된 톤을 유지합니다.

준비하는 방법은 3가지 — 편한 걸 고르세요.

  • 방법 A. 직접 만들기

    클로드 채팅창에 원하는 색상·폰트를 말하거나, 레퍼런스 이미지를 첨부해 이거 분석해서 design.md로 만들어줘라고 요청하세요. 한 번 잘 만들면 계속 재사용할 수 있어요.

  • 방법 B. 기존 템플릿 쓰기

    oh-my-design.kr/builder — 카카오·토스·네이버·애플 등 기업 디자인이 올라와 있어요.

    원하는 색을 누르고 export → download 클릭하세요.

  • 방법 C. 그냥 말로 하기 (3단계에 가서)

    파일 없이 채팅창에 깔끔한 블랙&화이트 스타일로 만들어줘처럼 입력해도 됩니다.

③ 규격·옵션

항목정할 것예시
사이즈가로 / 세로세로 1080×1920 (릴스) / 가로 1920x1080 (유튜브)
자막어떤 언어한글
음성넣을지 / 언어영어 TTS

3. AI한테 이렇게 말하면 됩니다.

클로드 코드 채팅창에 이렇게 적어주세요. 

[원고 파일명] 을 내용으로 한 영상 만들어줘.
영상 규격은 [가로 1920x1080 / 세로 1080x1920] 이야.
디자인은 [디자인.md]참고해.
자막은 [한글]로 달고, 음성 TTS는 [영어]로 달아.
효과음도 버튼·요소 등장할 때 넣어주고, 배경음악도 어울리는 걸로 깔아줘.

채울 때 팁:

  • 디자인 — "claude.md 디자인으로" / "깔끔한 미니멀로, 글래스모피즘 스타일로" 처럼 적으세요.
  • 원고 — 파일이 폴더에 없는 경우 그냥 내용을 채팅창에 적으세요. (내용도 없으면 "이 주제로 대본도 네가 써줘"도 가능)
  • 자막·음성 — 한글 자막 가능, ai 음성 가능 (한국어 음성 지원X)

4. 결과물

  • 길이: 약 80초
  • 자막: 한글 / 음성: 영어 TTS / 효과음·배경음악 포함
