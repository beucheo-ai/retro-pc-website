2️⃣ 노션 연결 (MCP)

정리한 내용을 클로드가 바로 내 노션 페이지에 써줘요. 복붙할 필요 없이 끝까지 자동화돼요.

🔧 설치하는 법

클로드 코드에 목령어 한 줄로 등록하고, 노션 로그인으로 한 번 "허용"하면 끝. API 키 발급 같은 건 필요 없어요.

1) 터미널에 명령어 입력 (클로드 코드를 켜는 그 터미널에)

claude mcp add --transport http notion https://mcp.notion.com/mcp

→ Added HTTP MCP server notion 이라고 뜨면 등록 성공

2) 클로드 코드 안에서 인증

  • 채팅창에 /mcp 입력 → 목록에서 notion 선택 → Authenticate(인증)
  • 브라우저에 노션 로그인 창 뜨면 허용(Allow access) 클릭

3) 확인

  • 다시 /mcp 쳐서 notion이 connected로 뜨면 완료 🎉
💡 이 방식은 로그인(OAuth)이라 페이지마다 따로 연결할 필요 없이, 내 워크스페이스 통째로 쓸 수 있어요.

💡 사용하는 법

기존 노션 페이지를 수정하거나, 새로 정리한 내용을 노션에 바로 저장하고 싶을 때 쓰면 돼요.
