이 가이드는 유튜브 링크를 붙여넣으면 Gemini AI가 영상을 분석하고, 자동으로 인스타그램 카드뉴스를 생성해주는 로컬 웹 프로그램을 만드는 전 과정을 담고 있습니다.

준비물 : VS Code, 클로드 코드 (유료계정), 제미나이 api 키

→ VS Code에 클로드 코드를 연동하고, 클로드 코드 채팅창에서 대화하며 프로그램을 만드는 프로세스입니다.

⚙️ 1단계 : 사전 준비

1. VS 코드

1-1. Visual Studio Code 설치 및 클로드 코드 추가

VS 코드 다운로드 링크 : https://code.visualstudio.com/download
VS 코드 실행 후 좌측 네모 아이콘 클릭 → 클로드 코드 설치하기 → 클로드 유료 계정 로그인까지 해주세요.

1-2. 클로드 코드 실행하기

💡2단계에서 이 클로드 코드 채팅창을 사용할 거기 때문에 이 화면에서 대기해주세요.

1-3. Python 버전 확인

터미널 열기
터미널에 아래 명령어를 입력하세요
python3 --version

3.10 이상이면 OK. 낮으면 python.org에서 최신 버전 설치.

2. Gemini API 키 발급

  1. Google AI Studio 접속
  2. 좌측 하단 Get API key 클릭
  3. Create API key → 키 복사해서 안전한 곳에 저장
⚠️ API 키는 외부에 노출되지 않도록 주의하세요. 프로그램 실행 시 웹 UI에서 직접 입력하는 방식이라 코드에 저장하지 않아도 됩니다.

📦 2단계 : 프로그램 제작 및 실행

1. 클로드 코드로 프로그램 제작

💡클로드 코드 채팅창에 다음 공식을 이용해서 질문하세요.

[만들고 싶은 것] + [내가 생각하는 구조] + [가진 자산] + "제안해줘 / 질문있으면 해”

→ 클로드가 모호한 부분을 스스로 가정하여 제작하지 않고, 사용자에게 질문하며 프로그램을 정교하게 설계하도록 유도하는 공식입니다.

💡이 예시 템플릿을 다운로드 한 뒤 아래 내용과 함께 첨부해주세요.

“나중에 웹에서 따로 입력할 거기 때문에 코드에는 안넣을 거야”

💡
📌 카드뉴스 html 템플릿이란?

카드뉴스 디자인을 코드로 구현한 것으로, AI가 구조를 쉽게 인식할 수 있어 제작 속도가 크게 빨라집니다. 본 프로그램은 HTML 기반의 디자인 템플릿을 바탕으로 작동합니다.

기본 제공해 드린 예시 템플릿 외에도, 나만의 맞춤형 템플릿을 만들어 사용해 보세요!

HTML 디자인 템플릿 제작 방법은 [카드뉴스 완전정복] 전자책 14~17p에 상세히 안내되어 있으며, 더 다양한 템플릿은 [공유자료] → [템플릿] 폴더에서 확인할 수 있습니다.


▶ 전자책 바로가기: https://www.latpeed.com/products/k0XjI

2. 터미널에 명령어 입력하기

💡 클로드가 안내한 명령어를 VS 코드 터미널에 입력해 주세요. 명령어는 한줄씩 순서대로 [복사+붙여넣기] 엔터 누르시면 됩니다.
💡모르겠을 땐 클로드에게 “서버 띄워줘” 라고 입력하세요.
⚠️ 빨간색 텍스트가 뜬다면 (오류) 그 내용 전체 복사해서 클로드에게 입력하세요.

실행 명령어 전부 입력시 터미널에 아래처럼 뜨면 성공입니다. 

* Running on http://127.0.0.1:5000 (또는 localhost:8000)

웹으로 이동해 "http://127.0.0.1:5000"을 입력하면 프로그램이 뜹니다!

📦 3단계 : 프로그램 보완하기

1. 프로그램 보완하기

💡 저는 프로그램을 실행해보며 몇 가지 보완할 부분을 발견했고, 클로드 코드 채팅창에 보완할 점들을 입력했습니다.
💡
채팅 예시 : 카드뉴스 스크립트 제작시 내가 적은 규칙을 바탕으로 제작돼야 해. 스크립트 규칙 파일을 만들어줘. 그리고 그걸 읽고 만들 수 있게 경로를 수정해줘. 스크립트 규칙은 내가 직접 적을게.
📌 Tip : 여러분도 한번 실행해보고, 추가하고 싶은 기능이 있다면 클로드 코드 채팅창에 입력하면 됩니다.
