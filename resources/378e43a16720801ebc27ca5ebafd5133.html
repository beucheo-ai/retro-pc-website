🔤 폰트 설정

HyperFrames는 외부 CDN 폰트를 불러오지 못하기 때문에 폰트 파일을 작업 폴더에 넣어줘야 합니다. (ttf, otf, woff2 포맷 상관없고, index.html 코드 속 폰트명,파일경로가 작업 폴더와 일치해야합니다.)

<html 코드 속 폰트 관련 부분>

<style>

        @font-face {
            font-family: 'GmarketSansTTFBold';
            src: url('fonts/GmarketSansTTFBold.ttf') format('truetype');
            font-weight: 700;

→ fonts/GmarketSansTTFBold.ttf 인 경우

파일 목록 예시 

바탕화면/
└── 영상변환/
		└── fonts/
		└── GmarketSansTTFBold.ttf

→ 이렇게 되어있어야 합니다.

폰트 추가하는 과정 (예 : Pretendard 사용 시)

1. 폰트 파일 다운로드

아래 링크에서 최신 릴리즈 zip 다운로드 후 압축 해제

https://github.com/orioncactus/pretendard/releases

web/static/ 폴더 안의 woff2 파일을 영상변환/fonts/ 폴더에 복사

필요한 파일:

fonts/
├── Pretendard-Regular.woff2
├── Pretendard-Medium.woff2
├── Pretendard-Bold.woff2
└── Pretendard-ExtraBold.woff2

파일 목록 예시

바탕화면/
└── 영상변환/
		└── fonts/
		├── Pretendard-Regular.woff2
		├── Pretendard-Medium.woff2
		├── Pretendard-Bold.woff2
		└── Pretendard-ExtraBold.woff2
