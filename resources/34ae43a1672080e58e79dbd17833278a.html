직접 찾아볼 필요 없이, 클로드가 아침마다 뉴스 정보를 정리해서 보내준다면?

매일 아침 반복되는 검색 시간은 줄이고, 인사이트는 더 깊게 가져가세요.

STEP 1. 연동하기: 외부 앱 연결

클로드가 외부의 정보를 가져오고(Exa), 메시지를 보낼 수 있도록(Slack) 통로를 열어줍니다.

  • ☐ 커넥터 메뉴 접속: 클로드 좌측 옵션 → Customize(사용자지정) → Connectors(커넥터) → (+)버튼 → 커넥터 둘러보기
  • ☐ Slack 추가: 요약된 정보를 받아볼 메신저 계정 연결
  • ☐ Exa 추가: 기업의 실시간 IR 자료와 최신 웹 정보를 검색하기 위한 엔진 연결 (별도 회원가입 불필요)

STEP 2. 경로 설정: 정보가 담길 '그릇'

정보를 수신할 채널과 검색 도구를 최종 세팅합니다.

1. 슬랙(Slack) 세팅

  • 워크스페이스 내에 전용 채널을 생성합니다. (예: #news, #ai-news)
  • 클로드 앱이 해당 채널에 메시지를 보낼 수 있도록 권한을 허용합니다.

2. Exa 검색 도구 확인

  • Exa는 단순 검색을 넘어 '기업 보도자료'나 '특정 날짜 뉴스'를 필터링하는 데 최적화되어 있습니다.

STEP 3. 자동화 설정: 스케줄링

클로드가 스스로 일하게 만드는 핵심 단계입니다.

  1. 메뉴 이동: Co-work(코워크) → Scheduled(예정됨) 클릭
  2. 새 작업 생성
    • 이름: 일일 뉴스 브리핑 (원하는 이름)
    • 빈도: 매일(Daily) / 오전 8:30 (원하는 시간)
  3. 프롬프트 입력: 아래의 템플릿 프롬프트를 복사하여 붙여넣으세요.
  4. 즉시 실행으로 테스트 먼저 : 예약된 작업이 만들어졌다면 “즉시 실행”을 누른 뒤 좌측 “Scheduled”에 들어가 “모든 예약된 실행에 대해 허용” 버튼을 눌러주시길 바랍니다.
    • 만약 실행될 때마다 허용 버튼이 쓴다면 “커넥터 → 해당 도구의 도구권한 → 항상 허용”으로 바꿔주세요.

STEP 4. 핵심 프롬프트 (템플릿)

주식 정보 받아보기
  • ⚠️ 참고용 정보로만 활용해 주세요. 데이터의 정확성을 위해 반드시 원문(출처 링크)을 교차 검증하시기 바랍니다.
  • 주식정보 프롬프트를 채팅창에 입력 후 본인 종목으로 교체해서 사용하세요. 교체하는 방법은 아래 작성예시를 참고하세요.
    • 작성예시 ✍🏻 :
      💡
      “다음 프롬프트를 [종목명]으로 내용 교체해줘(분석대상, 전방산업, 공급망). 마크다운 블록형식으로 출력해.

      ---

      주식 프롬프트 붙여넣기 “

## Slack 작성 규칙 (매우 중요):
* 모든 텍스트의 굵기 강조는 별표 하나(`*`)를 사용한다. (예: `*텍스트*`) — 별표 두 개(`**`)는 Slack에서 렌더링되지 않으므로 절대 사용 금지.
* 각 항목(AI 한줄 평 / 영향 / 수급 / 가이던스 / 목표가 / 출처) 사이에는 반드시 **빈 줄 1개**를 넣어 가독성을 확보한다.
* 각 종목 블록은 반드시 `━━━━━━━━━━━━━━━━━━━━` 구분선으로 시작하고 끝내며, 종목과 종목 사이에는 **빈 줄 2개**를 둔다.
* 각 항목의 라벨(예: `*수급*`)과 내용은 줄바꿈으로 분리한다. 한 줄에 라벨과 내용을 함께 쓰지 않는다. (단, `*영향*`과 `*목표가*`처럼 짧은 값은 한 줄 허용)
* URL은 Slack mrkdwn 링크 형식 `<URL|링크>`으로 감싸서 깔끔하게 표시한다.
* 4개 종목 분석 + 매크로 섹션을 반드시 하나의 단일 Slack 메시지로 전송한다.
