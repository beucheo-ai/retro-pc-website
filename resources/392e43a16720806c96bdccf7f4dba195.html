1. 무료 이용권?

  • "클로드가 뿌리는 무료 이용권" — 바로 게스트 패스예요.
  • Claude Pro는 원래 월 $20짜리 유료 구독이라, 돈을 내야 최신 모델과 Claude Code를 쓸 수 있어요.
  • 그런데 이 이용권만 있으면 7일 동안 그 유료 기능을 통째로 공짜로 써볼 수 있어요.
  • 앤스로픽이 "일단 써보고 결정하라"며, 기존 구독자가 주변에 나눠줄 수 있게 풀어둔 체험권이에요.
  • 그래서 지인에게 받거나, 커뮤니티에 공유된 링크만 구하면 누구나 무료로 시작할 수 있어요.

2. 나도 받을 수 있나요? (조건 확인)

가장 먼저 이거부터 확인하세요. 조건은 딱 하나예요.

  • ✅ 지금까지 Claude에 유료 결제를 한 번도 안 해본 계정 → 받을 수 있어요
  • ❌ 예전에 Pro나 Max를 결제해본 적 있는 계정 → 아쉽지만 받을 수 없어요
💡 결제 이력이 있는 계정이라면, 새 이메일로 계정을 새로 만들어서 받는 방법도 있어요.

3. 이용권 구하는 법

🙋‍♀️ 방법 1. 지인에게 요청하기 (가장 빠름)

  • 주변에 Claude Pro나 Max를 유료 구독 중인 지인이 있다면, 이용권을 나눠달라고 요청하세요.
  • 지인은 아래 방법으로 링크를 뽑아서 보내줄 수 있어요.
    • Claude Code에서 /passes 명령어 입력
    • 데스크톱 앱: 설정 → Claude Code / Cowork 탭에서 링크 확인
    • 모바일 앱: [Claude를 선물하세요] 또는 [선물상자 아이콘]에서 확인

🔍 방법 2. 커뮤니티에서 공유 링크 찾기

주변에 구독자가 없다면, 사람들이 공유해둔 링크를 활용하세요.

  • 구글에 이렇게 검색해보세요
    • "클로드 무료 이용권"
    • "클로드 무료 일주일"
    • "클로드 게스트패스"
  • 커뮤니티에서도 초대 링크가 자주 공유돼요
    • Reddit (r/ClaudeAI)
    • X(트위터), 스레드
    • 디스코드 커뮤니티

4. 이용권 사용하는 법

링크를 구했다면 이 순서대로 하면 돼요.

  1. 받은 이용권 링크를 클릭해서 접속
  2. 유료 결제를 한 적 없는 계정으로 로그인
  3. 결제 화면으로 넘어가면, 금액이 0원(무료)으로 표시되는지 확인 → [무료로 시작하기] 클릭
  4. 결제 수단(카드) 등록 → 여기까지 하면 7일 무료 체험 시작!
⚠️ 카드 등록은 필수예요. 대신 체험 기간(7일) 동안은 실제로 결제되지 않아요.

5. 뭘 쓸 수 있나요? (혜택)

  • 7일 동안 Claude Pro 전체 기능 무료 이용
  • 최신 모델 전부 (Opus · Sonnet · Haiku) 사용 가능
  • Claude Code + Cowork까지 포함

6. 유의사항 ⚠️

  • 결제 수단(카드) 등록은 필수 — 단, 체험 중엔 결제 안됨
  • 7일이 지나면 자동으로 월 $20 유료 결제로 전환돼요
  • 계속 쓸 게 아니라면 결제일 전에 반드시 구독을 취소하세요
  • 유료 결제 이력이 있는 계정은 이용할 수 없어요
