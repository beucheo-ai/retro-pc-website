📘 옵시디언 + 클로드 셋업 가이드 (초보용)

흩어진 메모를 클로드 코드에 연결해, 나만의 슬래시 명령어로 매일 공부·정리·아이디어 발굴을 자동화하는 초보용 셋업 가이드.

0. 시작 전에

0-1. 이 가이드로 결국 뭘 하게 되나

  • 옵시디언(메모장 앱)에 평소 생각과 아이디어를 적고, 흩어져 있던 메모들(노션,아이폰 메모)도 여기에 모은다.
  • 그 메모를 클로드가 직접 읽게 연결한다.
  • 클로드가 내 메모를 바탕으로 공부시켜주고, 정리해주고, 아이디어를 뽑아준다.
핵심 원리: “클로드에게 주는 정보(내 메모)의 질이, 클로드가 해줄 수 있는 일을 결정한다.”

0-2. 필요한 것 체크리스트

  • ☐ Obsidian (무료 메모 앱)
  • ☐ 클로드 코드 (클로드 유료 계정) ⚠️ 클로드 코드는 VS 코드 환경에서 사용합니다.
  • ☐ VS 코드

0-3. 용어 3개만 알고 가기

용어쉽게 말하면
Obsidian마크다운으로 메모를 적고 서로 링크하는 메모 앱
클로드 코드내 컴퓨터/파일을 다룰 수 있는 클로드. 명령어를 만들어 쓸 수 있음
MCPObsidian과 클로드를 이어주는 “연결 통로”

1. 설치 단계 (한 번만 하면 끝)

1-1. Obsidian 설치 + 첫 메모(Vault) 만들기

  1. obsidian.md 에서 다운로드·설치
  2. 실행 → Create new vault 로 새 보관함(폴더) 생성
  3. 메모 하나 만들어 아무 내용이나 적어보기
Vault = 내 메모들이 모인 폴더. 클로드가 읽을 “재료 창고”.

1-2. 클로드 코드 설치

  • VS코드를 설치하고 클로드 코드를 추가한 뒤 로그인한다.
    VS Code와 클로드 코드 세팅법. (▶ 클릭하세요)

    ⚙️ 사전 준비

    1. VS 코드

    1-1. Visual Studio Code 설치 및 클로드 코드 추가

    VS 코드 다운로드 링크 : https://code.visualstudio.com/download
    VS 코드 실행 후 좌측 네모 아이콘 클릭 → 클로드 코드 설치하기 → 클로드 유료 계정 로그인까지 해주세요.

    1-2. 클로드 코드 실행하기

1-3. 둘 연결하기 (MCP 연결)

  • 클로드 코드가 내 Obsidian Vault 폴더를 읽을 수 있도록 연결한다.
  • 가장 쉬운 방법: 클로드 코드에게 직접 부탁하기 (채팅창에 입력)
    > “내 Obsidian Vault 폴더를 읽을 수 있게 MCP 연결을 설정해줘.”

1-4. 연결 잘 됐는지 확인하는 법

  • 클로드 코드에게 시켜본다:
    > “내 Vault에 있는 메모 목록을 보여줘.”
  • 메모 제목들이 나오면 연결 성공.

2. 첫 메모 쌓기 (도구의 연료)

2-1. 왜 메모부터 써야 하나

  • 클로드는 내 메모에 있는 내용만 다룰 수 있다.
  • 메모가 비어 있으면 → 클로드도 해줄 게 없다.
  • 많이 쓸수록 도구가 똑똑해진다.

2-2. 오늘 당장 쓸 메모 3개 예시

  • 오늘 한 일 / 배운 것 (데일리 노트)
  • 요즘 관심 있는 주제 1개와 그 이유
  • 내 업계에서 요즘 뜨는 트렌드/키워드

2-3. 하루 5분 메모 습관 만들기

  • 완성된 글이 아니어도 OK. 토막 생각이면 충분.
  • 정해진 시간(예: 자기 전 5분)에 그날 생각을 적는다.

2-4. 이미 쓴 메모 끌어오기 (아이폰 메모·노션) - 부체오 TIP

아이폰 메모나 노션에 쌓아둔 글이 있다면 그걸 끌어와 바로 “연료”로 쓰기.
  • 아이폰 메모 가져오는 법 : https://obsidian.md/ko/help/import/apple-notes
  • 노션 가져오는 법 : https://obsidian.md/ko/help/import/notion
평소 쓰던 글을 그대로 활용할 수 있다.

3. 커리어 활용 흐름

“조용히 공부만 하면 안 알아준다. 공유해야 기회로 이어진다.”

3-1. 트렌드 하나 골라 클로드에게 배우기

  • 내 업계에서 지금 뜨는 트렌드 1개 선택
  • 프롬프트 예시:
💡
내 분야의 [트렌드]에 대해, 내 학습 스타일에 맞춰 가르쳐줘.

3-2. 내 부족한 스킬 → 7일 공부 체크리스트 받기

  • (선택) Obsidian 플러그인 Smart Connections 설치 → 메모 속 맥락·패턴을 더 잘 찾음
  • 프롬프트 예시:
💡
내 업계 5대 트렌드를 기준으로, 내가 부족한 스킬 목록을 뽑아줘. 그리고 그걸 7일 심화 학습 체크리스트로 만들어줘.
💡 부체오 TIP — 체크리스트를 매일 아침 카톡으로 받기 (▶ 클릭하세요)

클로드 코드 루틴, playmcp 활용

4. 명령어 만들어 보기

4-1. “명령어는 다운로드가 아니라 말로 만든다”

  • 클로드 코드에게 “이런 명령어 만들어줘” 라고 말하면 → 자동 생성·저장됨.
  • 다음부터는 /출근 처럼 입력만 하면 실행된다.

4-2. 따라하기: /출근 명령어 직접 만들기 (제일 쉬운 것)

  1. 클로드 코드에 입력:
💡
‘/하루’ 라는 슬래시 명령어를 만들어줘. 내 데일리 노트, 일정, 할 일 목록을 읽고, 이번 주에 중요하다고 한 것 기준으로 오늘의 우선순위 플랜을 만들어주는 기능이야.
  1. 클로드가 명령어를 만들어 저장.

4-3. 만든 명령어 실행해 보기

  • 그냥 /출근 입력 → 오늘의 플랜이 나오면 성공.

4-4. 추천 순서 — 초보가 먼저 만들면 좋은 명령어 3개

  1. /출근 — 아침 계획 (가장 쉬움)
  2. /퇴근 — 하루 마감 정리
  3. /아이디어 — 메모 기반 아이디어 뽑기
한 번에 다 만들지 말고, 하나 만들어 써보고 → 좋으면 다음 것 순으로 늘린다.

5. 하루 루틴으로 굳히기

5-1. 아침 / 저녁

  • 아침: /출근 로 하루 계획
  • 저녁: /퇴근 로 한 일·배운 점 정리

5-2. 주 1회

  • /승격 — 데일리 노트 속 아이디어를 독립 노트로 승격
  • /아이디어 — 쌓인 메모에서 새 아이디어·프로젝트 찾기

부록. 복붙용 프롬프트 모음 (12개 명령어)

사용법: 아래 문장을 복사 → [ ] 빈칸을 내 내용으로 바꿔 클로드에 입력.
명령어복붙용 프롬프트
/상황‘/상황’ 라는 슬래시 명령어를 만들어줘. 내 Vault를 읽고 현재 상황을 요약해주는 기능이야. 진행 중인 프로젝트, 최근 회고, 지난 7일간 언급한 우선순위를 포함해.
/출근‘/하루’ 라는 슬래시 명령어를 만들어줘. 내 데일리 노트, 일정, 할 일 목록을 읽고 이번 주에 중요하다고 한 것 기준으로 오늘의 우선순위 플랜을 만들어주는 기능이야.
/추적‘/추적’ 라는 슬래시 명령어를 만들어줘. [주제] 아이디어를 내 Vault 전체에서 추적해서 언제 처음 등장했고, 어떻게 발전했고, 지금 무엇과 연결돼 있는지 보여주는 기능이야.
/연결‘/연결’ 라는 슬래시 명령어를 만들어줘. 내 Vault에서 [주제 A]와 [주제 B]의 연결을 찾아, 둘을 잇는 노트와 패턴을 보여주는 기능이야.
/분신‘/분신’ 라는 슬래시 명령어를 만들어줘. 내 Vault를 바탕으로 [질문]에 내가 어떻게 답할지, 내 말투를 쓰고 관련 노트를 인용해서 보여주는 기능이야.
/반박‘/반박’ 라는 슬래시 명령어를 만들어줘. [주제]에 대한 내 노트를 검토해서 내가 어디서 모순되고, 어떤 가정이 틀렸을 수 있는지 짚어주는 기능이야.
/아이디어‘/아이디어’ 라는 슬래시 명령어를 만들어줘. 내 Vault에서 떠오르는 패턴을 스캔해서 만들 도구, 연락할 사람, 조사할 주제, 쓸 글에 대한 아이디어를 만들어주는 기능이야.
/승격‘/승격’ 라는 슬래시 명령어를 만들어줘. 최근 14일 데일리 노트를 스캔해서 독립 노트가 될 만한 아이디어를 찾아, 각각 핵심 주장·맥락·연결을 담은 파일로 만들어주는 기능이야.
/퇴근‘/퇴근’ 라는 슬래시 명령어를 만들어줘. 오늘 내가 한 일을 검토해서 진행 상황을 요약하고, 새로 떠오른 아이디어를 정리하고, 내일로 넘길 미완료 항목을 적어주는 기능이야.
/무의식‘/무의식’ 라는 슬래시 명령어를 만들어줘. 서로 무관한 노트들에 반복 등장하는 테마나 표현을 스캔해서, 내가 무의식적으로 향하고 있는 아이디어가 뭔지 알려주는 기능이야.
/발굴‘/발굴’ 라는 슬래시 명령어를 만들어줘. 내 Vault에서 프로젝트·에세이·제품이 될 만한 아이디어 군집을 찾아, 무엇이 떠오르고 있고 어떤 노트가 연결되는지 보여주는 기능이야.
/일정‘/일정’ 라는 슬래시 명령어를 만들어줘. 내 현재 프로젝트와 우선순위 기준으로 이번 주 일정을 제안하고, 중요하다고 말한 것과 실제 시간 쓰는 방식의 충돌을 표시해주는 기능이야.
