1. 클로드가 뭐예요?

클로드는 Anthropic이 만든 AI 어시스턴트예요. 쉽게 말하면, 무엇이든 물어볼 수 있는 매우 똑똑한 대화 상대예요.

🧠 이런 걸 할 수 있어요

- 글쓰기, 요약, 번역

- 코드 작성 및 디버깅

- 아이디어 브레인스토밍

- 문서 분석 및 정리

- 엑셀, Word, PPT 파일 생성

- 자동화

2. 주요 기능 소개

📁 스킬 (Skills)

스킬은 클로드가 특정 작업을 더 잘 수행하도록 돕는 지침 모음이에요. 마크다운 형식의 텍스트 파일로 되어 있고, 관련 작업이 필요할 때 클로드가 자동으로 꺼내서 씁니다.

📌 비유하자면?
요리사에게 특정 요리 레시피북을 쥐여주는 것과 같아요. 그럼 원하는 레시피를 쓰고 싶다면 레시피북을 우리가 직접 만들어서 클로드에게 주거나, 레시피북을 찾아서 제공해주면 되겠죠?
스킬 이름하는 일
📄 docxWord 문서 생성/편집
📊 xlsx엑셀 스프레드시트 생성
📑 pptx파워포인트 생성
📋 pdfPDF 읽기/생성
🇰🇷 hwpx한글(.hwp) 문서 생성
🎨 frontend-design고품질 웹 UI 생성
🎬 video-input영상 분석/요약
🛠️ skill-creator새 스킬 직접 만들기

예시

"이 데이터로 엑셀 파일 만들어줘"
"회의 내용 정리해서 Word 문서로 저장해줘"
"유튜브 영상 요약해줘" (영상 첨부)
스킬은 어떻게 만들까요?/ 사용할까요?
💡
  1. 채팅창에 “스킬을 만들고 싶어.” 라고 입력하면 클로드가 질문을 합니다. 같이 대화해서 맞춤 스킬을 제작하면 클로드가 코드 또는 다운로드 파일을 제공합니다.
  2. 클로드 좌측 상단 Customize (사용자 지정) → 스킬 클릭
  3. 스킬 만들기 → 파일이라면 ’스킬 업로드’, 코드라면 ‘스킬 지침 작성’란에 붙여넣어서 스킬을 저장해줍니다.

+ 부체오 TIP

  • 인스타그램이나 스레드 등에서 깃허브에 배포된 유용한 스킬들을 공유해주시는 분들이 많습니다. 눈에 보일 때마다 저장하셔서 ‘스킬 업로드’ 란에 md 파일을 업로드 하는 것도 좋아요!
  • 또는 채팅창에 ‘~ 목적이 있어서 ~ 스킬을 쓰고 싶은데 웹이나 깃허브에서 찾아줘.’ 라고 입력하시면 여러개를 찾아서 링크를 줍니다. 링크로 들어가서 다운받고 ‘스킬 업로드’ 에 업로드 하면 됩니다.

🔌 커넥터 (Connectors)

커넥터는 외부 서비스와 실시간으로 연결하는 도구예요. 스킬과 다르게, 실제 외부 API에 접근해서 데이터를 읽고 쓸 수 있어요.

⚠️ 스킬 vs 커넥터 차이

스킬은 "지침서" — Claude에게 방법을 알려주는 것

커넥터는 "팔다리" — Claude가 실제로 외부에 접근하는 것

커넥터할 수 있는 것
📁 Google Drive파일 읽기/쓰기
💬 Slack메시지 전송, 채널 읽기
📝 Notion페이지 생성/편집
📧 Gmail메일 읽기/작성
🎨 Figma디자인 파일 접근
📊 Google Sheets스프레드시트 연동

예시

"내 Google Drive에서 Q3 보고서 찾아줘"
"Slack #마케팅 채널에 공지 올려줘"
"이 내용을 Notion 페이지로 정리해줘"
커넥터는 어떻게 사용할까요?
  1. 클로드 앱 좌측 상단의 Customize 버튼 클릭 → 커넥터 섹션으로 이동 → 커넥터 둘러보기 클릭
  2. 연결할 도구 검색 (영문으로) → + 버튼 클릭→ 로그인 또는 연결 완료 후 커넥터 목록에 추가되었는지 확인
내 일에 적용하고 싶은데.. 어떤 커넥터를 써야 할지 모를 때?
  • 클로드에는 수많은 커넥터가 있어서, 처음에는 어떤 것을 골라야 할지 막막할 수 있습니다. 이럴 때는 클로드에게 추천을 요청하면 됩니다!

→ Claude가 화면 속 커넥터 목록을 분석해서 목적에 맞는 최적의 커넥터를 추천해줄 거예요!

💬 프로젝트 & 지침

프로젝트는 관련된 대화를 하나로 묶어 관리하는 공간이에요. 지침을 설정하면 해당 프로젝트의 모든 대화에 항상 적용돼요.

🗂️ 이럴 때 써봐요

- 특정 업무 전용 공간 만들기 (예: 마케팅 프로젝트)

- 항상 특정 말투나 형식으로 답변받고 싶을 때

- 반복적으로 쓰는 배경 정보를 매번 입력하기 싫을 때

프로젝트는 어떻게 사용하나요?
  1. 죄측 상단 projects 클릭 → new project 클릭해서 프로젝트 생성
  1. 지침과 파일

지침 : 마크다운 형식의 프롬프트를 붙여넣는 곳입니다. 넣어두면 클로드가 지침에 적힌 대로 행동합니다. 예를들어 지침에 ‘행동강령’, ‘역할’, ‘출력형식’을 적어두면 클로드가 그대로 이행합니다.

파일 : 클로드가 프로젝트에서 행동할 때 참고할 자료를 넣는 곳입니다. 마크다운 형식의 프롬프트 파일인 md 파일을 이곳에 넣어도 됩니다.

💡
🎨 사례 1: 앞으로 내가 내용만 넣으면 클로드가 카드뉴스를 만들게 하고 싶다.

지침 : 카드뉴스를 어떤 순서로 만들건지, 출력은 어떻게 할건지 (png 인지, html 코드인지), 규칙은 무엇인지 (폰트는 pretendard 만 사용해야한다 등등) 적힌 곳

파일 : 디자인 html 코드 또는 디자인 이미지를 첨부해둠 → 앞으로 프로젝트에서 카드뉴스를 만들 때 클로드는 파일 속 디자인을 참고함.

💡
📝 사례 2: 논문을 넣어두고 논문 분석 또는 질문을 하고 싶다.

지침 : 사용자가 채팅을 어떻게 입력하면 클로드가 어떤식으로 답변할건지 출력형식을 적어두고, 지어내서 대답하지 않도록 하게끔 규칙을 적어두는 곳

파일 : 논문을 첨부해두는 곳

스킬 vs 프로젝트 지침 — 헷갈리기 쉬운 둘의 차이
스킬프로젝트 지침
언제 로드?필요할 때만항상
컨텍스트 소모필요시만항상 소모
적용 범위전체 공통해당 프로젝트만
부체오 요약
  • 지침과 스킬 모두 마크다운 형식의 텍스트입니다. 그렇다면 둘이 뭐가 다를까요?
  • 스킬 : 마크다운 텍스트를 스킬 란에 한 번 저장해두면 클로드와 대화할 때마다 알아서 불러와져요. 따로 뭔가를 입력할 필요 없이, 관련 작업이 생기면 클로드가 스스로 판단해서 씁니다.
  • 반면 지침은 마크다운 텍스트를 프로젝트 안에 저장하는 거라, 그 프로젝트 안에서만 약속된 방식으로 움직여요. 그래서 이 둘을 잘 나눠 쓰면 꽤 편해지는데요, 어디서든 공통으로 쓰는 말투나 기본 규칙은 스킬에 넣어두고, 인스타그램 카드뉴스 제작처럼 특정 작업에서만 필요한 규칙은 프로젝트 지침에 따로 정리해두면 훨씬 깔끔하게 관리할 수 있어요.

STEP 1. 내 일 돌아보기

🧩 생각해볼 것들

- 글 쓰는 데 오래 걸리는 편인가요?

- 자료 조사나 정리에 시간이 많이 드나요?

- 회의록, 보고서, 이메일 작성이 부담스럽나요?

🏠 일상여행 계획, 레시피, 육아 아이디어

STEP 2. 직접 클로드에게 물어보기 💬

아직 어떻게 써야 할지 모르겠다면, 클로드에게 직접 물어봐요!

예시 질문 그대로 복사해서 써봐요 👇

"나는 [직업/역할]이야.
내 업무 중 [구체적인 작업]을 자주 하는데,
AI를 어떻게 활용하면 효율적으로 할 수 있을까?"

실제 예시

"나는 회사 마케터야.

  2. 답변이 마음에 안 들면 "일단 나한테 질문해봐(내 상황을 자세히 물어보게 유도)", "예시 제공해줘"처럼 수정 요청
  3. 원하는 결과가 나올 때까지 계속 대화!

STEP 4. 프롬프트로 만들어서 프로젝트에 저장해둬요 📌

나만의 맞춤형 AI 지침 만들기
  • 원하는 결과가 나왔다면? 다음부터는 이 대화처럼 AI가 그대로 행동할 수 있도록 '행동강령'이나 '규칙'이 담긴 ‘지침 프롬프트’를 써달라고 하세요.
    지금 대화한 걸 바탕으로 프로젝트용 지침 프롬프트를 만들어줘.
  • 프로젝트 지침에 복사해서 붙여넣기
    AI가 정리해준 프롬프트를 복사해 '프로젝트’ 생성 → 지침에 붙여넣어 저장하면, 다음부터는 프로젝트에 가면 긴 설명 없이 정보를 얻을 수 있어요.
