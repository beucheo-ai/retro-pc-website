노트북LM(NotebookLM) 이란?

"사용자가 제공한 소스만을 근거로 답변하는 AI 맞춤 비서"
어려운 내용을 이미지, 영상으로 쉽게 설명하고, 나만의 오디오를 만들어 이동 중에 학습하며, 내 자료 기반의 무한 퀴즈 생성이 가능합니다.

🏗️ 1. 노트북LM 기본 구조

노트북LM은 총 3개의 영역으로 구성되어 있습니다.

  1. [출처] 좌측 : 자료 첨부하는 곳
    • 학습의 기반이 되는 PDF, 웹 검색, 유튜브 자료를 저장
    • 외부 유출 걱정 없이 개인적인 자료나 미공개 문서 업로드 가능
  2. [채팅] 중앙 : AI와 채팅하는 곳
    • 제미나이와 채팅하듯 질문하고 답변을 얻는 영역
    • 첨부된 자료 내에서만 답변하므로 환각(Hallucination) 현상이 적음
  3. [스튜디오] 우측 : 다양한 결과물 생성
    • '출처'와 '채팅' 내용을 바탕으로 영상, 보고서, 인포그래픽, 슬라이드 등 생성 및 저장

🚀 2. 영역별 실전 사용법

[01] 출처 (Source)

업로드할 PDF가 없다면 '딥리서치' 전략을 활용하세요.

  • 웹(Web) 검색: 인터넷의 최신 정보를 검색하여 즉시 내 소스로 가져오기
  • 유튜브 링크: 유튜브 링크를 소스로 가져오면 AI가 내용을 분석하여 활용

[02] 채팅 (Chat)

질문을 넘어 '능동적 인출(Active Recall)'을 실천하는 4단계:

  1. 질문하기 (개념 문의) → 2. 답변 확인 (논리적 이해) → 3. 메모 저장 (클릭 필수!) → 4. 스튜디오 (우측 보관함)
💡 Tip: 채팅 내역은 새로고침 시 사라집니다. 중요한 답변은 반드시 메모에 추가를 눌러 스튜디오에서 확인하세요.

✍️ 3. [채팅 영역] 실시간 소통 & 문제 해결 프롬프트

공통 마법 공식: [맥락(C) + 상황(S) + 목적(P) + 출력 양식(F)]

💡 시험 출제 위원 모드 (자격증/수험생)
💡
"당신은 이 자료를 숙달한 시험 출제 위원입니다. 실제 시험 수준의 변별력을 가진 4지선다형 문제 100개를 생성해 주세요. 난이도는 상중하로 배분하고, 반드시 [문제/보기/정답/해설(소스 인용)] 형식을 유지하세요."
💡 1:1 지능형 튜터 모드 (대학생/교사)
💡
"사용자의 이해를 돕는 학습 파트너가 되어주세요. 제공된 자료에서 질문을 하나씩 던지고, 제가 답을 하면 정답 여부를 확인해 주세요. 만약 제가 모른다고 하면 정답을 바로 주지 말고 '소스 내의 힌트'를 주어 스스로 생각하게 유도해 주세요."

🎨 4. [스튜디오 영역] 기능별 맞춤 프롬프트

텍스트가 너무 많을 때 시각적/청각적 자료로 변환하여 학습 효율을 높입니다.
