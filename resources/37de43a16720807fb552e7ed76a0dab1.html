1. 개요

Claude + PlayMCP(카카오톡 나챗방)을 연동해서, 관심 종목의 최신 뉴스와 선행 리스크 신호를 매일 자동으로 카카오톡 나에게 채팅으로 받아보는 방법입니다.

2. 사전 준비

2-1. 클로드에 PlayMCP 연결

💡
  1. Claude의 설정 > 커넥터 로 이동합니다.
  2. 상단의 [커넥터 둘러보기]를 선택합니다.
  3. 디렉토리에서 PlayMCP를 검색하시거나, 목록에서 PlayMCP를 선택해 주세요.
  4. [연결]을 누르면 Claude와 PlayMCP 연동 과정이 시작됩니다.
  5. PlayMCP 가입 시 사용한 카카오 계정으로 로그인한 후 OAuth 인증 및 동의를 진행합니다.
  6. 연결이 완료되면 채팅에서 PlayMCP 도구함에 담은 도구를 바로 사용할 수 있습니다.

2-2. PlayMCP에서 카카오톡 나챗방 연동

바로가기 : https://playmcp.kakao.com/mcp/1
  1. [도구함에 추가]를 선택해 주세요.
  2. PlayMCP를 연결한 카카오 계정으로 로그인해 주세요.
💡

(선택) yfinance MCP — 더 정확한 주가/뉴스 데이터를 원하면 설치

더 정확한 주가 데이터, 애널리스트 추천 의견, 실적 발표 후 주가 흐름 등을 활용하고 싶다면 yfinance MCP를 추가로 설치하면 좋습니다.

설치 방법: 클로드 채팅창에 아래의 깃허브 링크를 제공하고 "설치 방법을 안내해줘"라고 요청하면 됩니다.
  • 깃허브: https://github.com/Alex2Yang97/yahoo-finance-mcp
💡
✍🏻 작성예시

https://github.com/Alex2Yang97/yahoo-finance-mcp 이거 보고 설치 방법 안내해줘.

3. 자동화 설정 (클로드 데스크톱 앱 실행)

  1. 좌측 메뉴 이동: Co-work(코워크) → Scheduled(예정됨) 클릭
  2. 새 작업 생성
    • 이름: Stock news briefing (원하는 이름)
    • 빈도: 매일(Daily) / 오전 7:00 (원하는 시간)
  3. 프롬프트 입력: 설명 아래의 빈칸에 프롬프트를 붙여넣으세요.
  4. 즉시 실행으로 테스트 먼저 : 예약된 작업이 만들어졌다면 “즉시 실행”을 누른 뒤 좌측 “Scheduled”에 들어가 “모든 예약된 실행에 대해 허용” 버튼을 눌러주시길 바랍니다.
    • 만약 실행될 때마다 허용 버튼이 쓴다면 “커넥터 → 해당 도구의 도구권한 → 항상 허용”으로 바꿔주세요.

→ 본 자동화 기능은 PC가 켜져 있는 상태에서만 작동합니다. PC가 꺼져 있을 경우, 전원이 켜진 직후에 자동으로 실행됩니다.

PC가 꺼져 있을 때 원격으로 실행하는 방법 → 클로드 코드 '루틴' 기능 활용
  1. 클로드 데스크톱 앱 실행
  2. 좌측 메뉴 이동: Code → 루틴 클릭 → 우측 상단 새 루틴 클릭 → 원격 선택
  3. 프롬프트 입력 : 지침란에 아래 제공된 프롬프트를 입력하세요.
    • 커넥터: Play MCP 추가
    • ⚠️ yfinance MCP처럼 로컬 개발 MCP는 루틴에서 추가가 불가합니다. 이 경우 Exa로 대체하는 것을 권장합니다.

주식 정보 프롬프트

yfinance MCP 설치 여부에 따라 선택해서 사용할 수 있는 2가지 프롬프트입니다
일반 검색용 프롬프트 (설치하지 않은 경우)
오늘 날짜를 기준으로 아래 5개 종목의 최신 뉴스를 검색하고, 카카오톡 나챗방으로 전송해주세요.

## 검색 조건
* 웹 검색을 사용하여 각 종목의 최신 뉴스를 조사할 것
* 종목명/티커가 제목 또는 본문에 직접 언급된 기사만 사용할 것 (섹터 전반 뉴스 중 해당 종목이 단순 언급만 된 경우는 제외)
* 각 종목마다 오늘 또는 최근 24시간 이내에 발행된 기사를 우선 사용할 것
* 오늘 기사가 없으면 가장 최근 기사를 사용하되, 날짜를 명시할 것
* 출처는 도메인명과 발행일로 표기 (메시지 글자수 절약을 위해 전체 URL 대신 도메인만 사용, 예: finance.yahoo.com)

## 선행 리스크 신호 점검 (각 종목별)
* 웹 검색으로 최근 1개월 내 애널리스트 등급/목표가 변경(상향/하향) 뉴스가 있는지 확인
* 최근 실적 발표가 있었다면, 발표 후 주가 반응(상승/하락) 관련 뉴스나 기사 언급을 확인 — 발표 후 하락 추세로 보도되면 "선행 신호"로 판단
* 뉴스에서 다음 항목 확인:
  - 어닝 서프라이즈/쇼크 여부 및 발표 후 시장 반응
  - 가이던스/전망치 하향 또는 상향
  - 매크로 이벤트(금리, 인플레이션 지표 등)로 인한 실적 무관 급락 여부
  - 주요 고객사·공급망 이슈, 수주/계약 관련 부정적 뉴스
* 위 신호 중 하나라도 해당되면 "⚠️감지(근거: 뉴스/등급변경/주가흐름)"로 표시하고 한 줄 설명 추가, 없으면 "특이사항없음"
